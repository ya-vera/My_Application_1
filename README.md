# Snake game (Android Kotlin)

Классическая игра "Змейка", разработанная для Android с использованием языка программирования Kotlin. 

"Змейка" — это популярная игра, в которой игрок управляет змейкой, увеличивающейся в длине по мере поедания еды.

Цель игры — направлять змейку, чтобы она съела как можно больше еды, не сталкиваясь со стенами или своим собственным телом.
